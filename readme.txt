ShippyPro Plugin for WooCommerce. Generate orders labels for more than 50+ carriers and connect more than 15+ marketplaces with just one single platform, Displays Live Shipping Rates based on the Shipping Address and Cart Content.

== Description ==

ShippyPro is a cloud software system management that allows you to integrate multiple sales channels and carriers with one click, import and manage all your orders in one simple universal interface

= Shows your carriers fares inside your WooCommerce checkout with the ShippyPro's plugin - without any additional cost! =

With ShippyPro plugin your WooCommerce checkout will show live carriers fares for each order and your customers will be able to choose their favourite carrier with LIVE pricing

When you create a shipping label in ShippyPro, the system will show you the shipping options choosen by your customer at the checkout


You and your customers will not overpay anymore for a remote area surcharge delivery!

== Installation ==

1. Upload the plugin folder to the '/wp-content/plugins' directory.

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Thats it, you can now configure the plugin.
